# Complete Demo Skill

## When To Use

Use when a local fixture needs a complete skill for smoke planning.

## Required Tools Or Inputs

- Node.js 20.
- Local fixture files.

## Side-Effect Boundaries

The workflow reads local files only.

## Approval Requirements

Ask before using real account data or external connectors.

## Examples

Run the local verification commands.

```sh
npm test
npm run smoke
```

## Validation Workflow

Run the local tests and capture terminal output.
